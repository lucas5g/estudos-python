## Requisítos 
* Crie um arquivo .env e defina as variáveis "OPENAI_API_KEY" e "GROQ_API_KEY"
* Python3

## Instalação 
```
sudo  apt install python3.10-venv 
python3 -m venv .venv 
source .venv/bin/active
pip install -r requirements.txt
```
## Rodar Aplicação
```
streamlit run src/Leitor_Pdf.py
```
